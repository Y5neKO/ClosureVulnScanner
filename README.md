## CVS - ClosureVulnScanner

目前正在开发中，广泛征求意见.


## 默认配置
`Python 3_8_0`  |  `PyCharm 2023.2.3`  |  `UTF-8`


## 目录描述
`core`: 核心功能目录

`exp`: 漏洞利用模块插件目录

`ez_poc`: 简单发包型poc验证模块插件目录

`finger`: 指纹信息目录

`log`: 程序日志目录

`poc`: 复杂poc验证模块插件目录
